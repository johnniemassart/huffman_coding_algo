The source code for this project is able to be compiled with JDK 17 from the terminal.

In order to run the program from the terminal, cd into the src folder wthin Lab1. Enter the commands:
(1) - javac HuffmanEncoding.java
(2) - javac HNode.java
(3) - javac HuffmanTreeMethods.java
(3) - java HuffmanEncoding (file input) (file output)
   * file input - "../input/(name).txt"
   * file output - "../output/(name).txt"
